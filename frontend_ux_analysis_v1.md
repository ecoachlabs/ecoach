## THE PRODUCT IN ONE SENTENCE

eCoach is a local-first, offline-capable desktop academic coaching OS for Ghanaian JHS students (BECE exam prep, ages 14-15), their parents, and super admins -- powered by a deep Rust intelligence engine that must feel invisible while its coaching feels deeply personal.

## THREE PRODUCTS IN ONE SHELL

| Surface | Aesthetic | Density | Tone |
|---|---|---|---|
| Student | Warm, energetic, game-inflected | Low -- one clear action at a time | Coaching, encouraging, never patronizing |
| Parent | Calm, premium, "private banking" | Medium -- interpreted summaries | Clinical, confident, plain-language |
| Super Admin | Dense, operational, developer-tools | High -- tables, pipelines, queues | Precise, data-oriented, throughput-focused |

## RECOMMENDED BUILD ORDER

Phase 0: Foundation (design system + shell)
Phase 1: Coach Hub + Session Player
Phase 2: Assessment + Diagnostics
Phase 3: Progress + Visualization
Phase 4: Content + Admin
Phase 5: Advanced Modes + Games
Phase 6: Polish + Accessibility
