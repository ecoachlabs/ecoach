# Screen Pattern Rebuild Kit

This file exists so the frontend rebuild can stay aligned with backend contracts.

## Core Screen Patterns

- dashboard with summary cards plus action rail
- indexed list with filters and detail panel
- session runner with timer, prompt, response area, and completion footer
- report view with summary, evidence breakdown, and recommendations
- admin workspace with queue table plus review drawer
- recovery view with audit status, backup actions, and snapshot export

## Contract Rule

Every new screen should name the commands it depends on before visual polish begins.
