4. Make it relationship-first, not page-first
This is where most glossaries fail.

A smart glossary must understand relationships between entries.

For example:

Area of triangle

depends on: base, height, multiplication, half

related to: area, triangle, rectangle

confused with: perimeter of triangle

appears in: mensuration questions

tested in: worked examples, word problems, direct recall

Photosynthesis

depends on: chlorophyll, sunlight, carbon dioxide, water

contrasted with: respiration

appears in: plant nutrition

often confused with: plant respiration

linked formula: word equation / chemical equation

So every entry needs a graph layer:

EntryRelationship
- from_entry_id
- to_entry_id
- relationship_type
    (depends_on, part_of, opposite_of, confused_with, example_of,
     formula_for, derived_from, used_in, tested_with, prerequisite_for,
     simpler_than, deeper_than, synonym_of)
- strength_score
- explanation
This changes the glossary from a static library into a knowledge network.

10. How Glossary Lab fits into the wider learning journey
This is important.

Glossary Lab should not be isolated. It should feed and be fed by the rest of the app.

Connected to Teach Mode
When a student learns a topic, glossary terms for that topic appear beside it.

Connected to Question Review
When a student misses a question, linked glossary items are recommended.

Connected to Memory Mode
Weak glossary terms enter memory rescue and spaced review.

Connected to Past Questions
Glossary items show how they appear in past exams.

Connected to Library Shelves
Saved glossary items become personal study shelves.

Connected to Audio Revision
Glossary Radio becomes one of the revision streams.

This makes it a real subsystem.

11. What truly makes it unique
Most glossaries help students read terms.

Your Glossary Lab should help them:

retrieve terms

hear terms

type terms

rebuild formulas

separate confusing concepts

recognize hidden concepts in questions

learn related items together

move from definition to application

That is what makes it unique.

12. The standout feature I would emphasize
If I had to choose the one combination that gives maximum value, I would build this path:

Learn → Listen → Link → Test → Transfer
For each term or bundle:

read it

listen to it

see what it is related to

test recall

test recognition in context

test use in a real question

That sequence is powerful and rare.
